Subtopic 7.1: Hardware

7.1.1 - Understand the features and use of different types of physical computers

This section describes four main categories of physical computers: personal computers, mobile devices (smartphones & tablets), servers, and embedded devices. For each type we'll cover features, typical uses, and advantages / limitations.

Personal Computers (PCs)

Features:

  • General-purpose machines available as desktops or laptops.
  • Key components: CPU, RAM, storage (HDD/SSD), GPU (optional), input/output devices (keyboard, mouse, display).
  • Runs desktop operating systems (e.g., Windows, macOS, Linux).
  • Often upgradeable (desktop more so than laptop).
  • Moderate to high computing power suitable for multitasking and heavier applications.

Uses:

  • Productivity: documents, spreadsheets, presentations.
  • Creative work: photo/video editing, design, software development.
  • Entertainment: gaming, streaming media.
  • Education, research, and general web browsing.

Advantages:

  • Flexible and powerful for many tasks.
  • Wide software compatibility.
  • Upgradeable hardware extends useful life.

Limitations:

  • Less portable than mobile devices (desktops).
  • Laptops trade off upgradability and repairability for portability.
  • Higher power consumption for high-performance models.

Mobile Devices (Smartphones & Tablets)

Features:

  • Portable, battery-powered devices with touchscreens.
  • Mobile operating systems (e.g., Android, iOS).
  • Integrated sensors: GPS, accelerometer, camera, microphone.
  • Built-in wireless connectivity (Wi-Fi, cellular 4G/5G, Bluetooth).
  • Often have system-on-chip (SoC) designs—compact and energy efficient.

Uses:

  • Communication: voice calls, messaging, video calls.
  • On-the-go internet access, social media, and apps.
  • Navigation, photography, media consumption.
  • Light productivity: email, notes, calendar, cloud apps.

Advantages:

  • Highly portable - always with the user.
  • Fast access to information and location-aware services.
  • Large ecosystem of mobile apps.

Limitations:

  • Less raw processing power than high-end PCs.
  • Smaller screen and text entry constraints.
  • Limited upgradeability and shorter hardware lifespan for some models.

Servers

Features:

  • Designed to provide services to multiple users or other computers over a network.
  • Server-grade hardware: multi-core CPUs, large memory capacity, redundant storage (RAID) and power supplies.
  • Optimized for reliability, availability, and scalability.
  • Typically run server operating systems and are housed in data centres or server rooms.
  • Often operate 24/7 and include advanced cooling and remote management features.

Uses:

  • Hosting websites, applications, and databases.
  • File and print services, authentication, and directory services.
  • Cloud infrastructure, virtualization, and container platforms.
  • Big data processing, enterprise applications, and backend services for mobile/web apps.

Advantages:

  • High performance and capacity for many simultaneous users.
  • Designed for reliability and uptime.
  • Scalable to meet growing demand (clustering, load balancing).

Limitations:

  • High cost for hardware, maintenance, and energy (cooling/power).
  • Requires specialised administration and security management.
  • Not intended for direct end-user interaction (no built-in UI for casual use).

Embedded Devices

Features:

  • Specialised computing components integrated into larger systems (e.g., microcontrollers, SoCs).
  • Designed for a dedicated function or small set of functions.
  • Optimised for low power, small size, and long-term reliability.
  • Often have limited or no direct user interface — they run firmware rather than full desktop OSs.
  • Common in Internet of Things (IoT) devices, sensors and real-time systems.

Uses:

  • Control systems: appliances, industrial controllers, automotive ECU (engine control units).
  • Wearables, smart home devices, sensors, and medical instruments.
  • Appliances and consumer electronics (TVs, microwaves, washing machines).
  • Specialised tasks where reliability and low power are essential.

Advantages:

  • Efficient and tailored to the task - minimal overhead.
  • Low power consumption and small physical footprint.
  • Cost-effective at scale for single-purpose functions.

Limitations:

  • Limited processing headroomnot suitable for general-purpose computing.
  • Often hard to update or patch (locked firmware or limited interfaces).
  • Security can be a concern if not designed with updates and protections in mind.

Summary

>
Type Key Features Typical Uses
Personal Computers General-purpose, upgradeable, moderate-to-high power Productivity, creative work, gaming, education
Mobile Devices Portable, touchscreen, battery-powered, sensors Communication, navigation, on-the-go apps, media
Servers High-capacity, reliable, network-focused, 24/7 operation Hosting, databases, cloud services, enterprise apps
Embedded Devices Specialised, low-power, firmware-driven Appliances, automotive systems, IoT sensors, controllers

Final notes:

Choosing the right type of physical computer depends on the task, required performance, need for portability, and constraints like power, cost, and reliability. Many modern systems combine multiple types (e.g., mobile clients connecting to server backends; embedded controllers reporting to cloud servers).

7.1.2 - Understand the features and use of different types of hardware devices

This section explains the different categories of hardware devices and their features, along with how and why they are used in computing systems.

Input Devices

Features:

  • Allow data and commands to be entered into a computer system.
  • Examples: keyboard, mouse, touchscreen, microphone, scanner.

Uses:

  • Typing and navigation (keyboard & mouse).
  • Capturing voice or images (microphone, camera, scanner).
  • Interactive input (touchscreen, stylus).

Output Devices

Features:

  • Display processed data from the computer to the user.
  • Examples: monitor, speakers, printers, VR headsets.

Uses:

  • Visual display of information (monitors, projectors).
  • Audio feedback (speakers, headphones).
  • Permanent copies of documents (printers).

Processors (CPU)

Key Features:

  • Number of Cores: Determines how many tasks can be processed simultaneously.
  • Clock Speed: Measured in GHz, higher speeds mean more instructions per second.
  • Cache Size: Small, high-speed memory close to the CPU for frequently used instructions/data.
  • Mobile Processors: Optimised for low power consumption and thermal efficiency in portable devices.

Uses:

  • General processing of instructions and data in all computing devices.
  • Multi-core processors for multitasking and parallel computing.
  • Mobile CPUs balance performance with long battery life in smartphones and tablets.

Main Memory

Types:

  • RAM (Random Access Memory): Temporary, volatile memory used to store active programs and data.
  • ROM (Read-Only Memory): Non-volatile memory storing essential instructions (e.g., BIOS/firmware).

Uses:

  • RAM allows quick read/write access for active processes.
  • ROM ensures that the system can boot and operate correctly.

Secondary Storage

Types:

  • Magnetic: Hard disk drives (HDDs), high capacity but slower and mechanical.
  • Solid State: SSDs, faster, no moving parts, more durable but costlier per GB.
  • Optical: CDs, DVDs, Blu-ray discs, used for media distribution and backups.

Uses:

  • Permanent storage of operating systems, software, and user data.
  • Data backup and archiving.

Motherboard

Features:

  • Main circuit board that connects all components of the system.
  • Houses CPU socket, RAM slots, expansion slots, and connectors.
  • Provides communication pathways (buses) between components.

Uses:

  • Central hub for all hardware components to interact.
  • Supports system expandability with additional cards and storage.

Graphics Processing Units (GPUs)

Features:

  • Specialised processors designed to handle complex graphics and parallel tasks.
  • Dedicated memory (VRAM) for high-performance rendering.

Uses:

  • Rendering images, animations, and video.
  • Running modern games smoothly.
  • Accelerating AI, scientific simulations, and cryptocurrency mining.

Network Interface Devices

Types:

  • PCI (Peripheral Component Interconnect): Expansion card interfaces for high-speed internal connections.
  • USB (Universal Serial Bus): Widely used interface for external devices and networking adapters.

Uses:

  • Enable computers to connect to wired and wireless networks.
  • Expand connectivity options through add-on cards or adapters.

Cooling

Types:

  • Air Cooling: Fans and heatsinks that dissipate heat using airflow.
  • Liquid Cooling: Circulates liquid coolant to absorb and transfer heat away from components.

Uses:

  • Prevents overheating and maintains performance.
  • Essential for high-performance CPUs and GPUs.
  • Liquid cooling often used in gaming PCs and servers.

Sensors

Features:

  • Hardware components that detect and measure environmental conditions.
  • Examples: temperature sensors, accelerometers, gyroscopes, fingerprint scanners.

Uses:

  • System monitoring (temperature, voltage).
  • Mobile devices (screen rotation, step counting, biometrics).
  • IoT and embedded systems (automation and environmental sensing).

Summary

Each hardware device plays a crucial role in enabling computer systems to input, process, store, and output data efficiently. Understanding their features helps in selecting the right components for specific tasks.
