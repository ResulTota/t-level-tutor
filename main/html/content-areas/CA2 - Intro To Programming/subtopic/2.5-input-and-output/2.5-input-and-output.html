2.5.1 - Understand how to implement input and output

Purpose: Capture data from the user and display results.

Keyboard Input:


name = input("Enter your name: ")
print("Hello, " + name)

Screen Output:


print("Welcome to TLevelTutor!")

Text File Input/Output:

Covered in section 2.5.2

2.5.2 - Know and understand the use of text files for input and output

Purpose: Store and retrieve data persistently using files.

Opening a File:

  • Read mode: r
  • Write mode: w
  • Append mode: a

Reading from a File:


# read() - reads the entire file
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

# readline() - reads one line at a time
with open("data.txt", "r") as file:
    line1 = file.readline()
    line2 = file.readline()
    print(line1, line2)

# readlines() - reads all lines into a list
with open("data.txt", "r") as file:
    lines = file.readlines()
    for line in lines:
        print(line.strip())

Writing to a File:


# write() - writes a single string
with open("data.txt", "w") as file:
    file.write("First line\n")
    file.write("Second line\n")

# writelines() - writes a list of strings
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("data.txt", "w") as file:
    file.writelines(lines)

Example: Reading and Writing Together


# Read names from a file, add greetings, write to another file
with open("names.txt", "r") as infile:
    names = infile.readlines()

with open("greetings.txt", "w") as outfile:
    for name in names:
        outfile.write("Hello " + name)

2.5.3 - Be able to interpret code using input and output

When reading code:

  • Identify input sources (keyboard or file)
  • Identify output destinations (screen or file)
  • Determine what data is read or written

Example:


file = open("numbers.txt", "r")
number = int(file.readline())
print(number * 2)
file.close()

2.5.4 - Be able to create code using input and output

Practice writing code to handle input from the keyboard and output to screen or file.

Example:


# Get user input and write to a file
name = input("Enter your name: ")
age = input("Enter your age: ")

with open("user_data.txt", "w") as file:
    file.write("Name: " + name + "\n")
    file.write("Age: " + age + "\n")

print("Data saved to user_data.txt")

2.5.5 - Be able to debug code using input and output

Common errors when handling input and output:

  • FileNotFoundError: Trying to read a file that does not exist. 
    
file = open("missing.txt", "r")  # ERROR
  • TypeError: Not converting input to the correct type. 
    
age = input("Enter age: ")  # returns string
print(age + 5)  # ERROR: cannot add str and int
  • File not closed: Forgetting to close a file can prevent saving or reading correctly. 
    
file = open("data.txt", "w")
file.write("Hello")
# missing file.close() may cause issues

Tip:

Use with open("filename", "r/w") as file: to automatically close files safely.
