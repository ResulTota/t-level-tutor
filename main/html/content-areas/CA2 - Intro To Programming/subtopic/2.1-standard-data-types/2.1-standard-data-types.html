2.1.1 - Know the definition of common data types and understand their purpose and when each is used

Integer

Definition: A whole number (positive or negative) with no decimal point.

When Used: Used when exact whole numbers are needed, such as counting items, indexing arrays, or loop counters.

Example: 10, -4, 2025

Float

Definition: A number that contains a decimal point, allowing for fractional values.

When Used: Used when precision is required, such as for measurements, currency calculations, or scientific data.

Example: 3.14, -0.5, 100.25

String

Definition: A sequence of characters (letters, numbers, symbols) enclosed in quotes.

When Used: Used for storing text, names, messages, or any alphanumeric data.

Example: "Hello", "12345", "A1B2C3!"

Boolean

Definition: A data type that can only have two possible values: True or False.

When Used: Used for logical decisions, conditions, and control structures in programs.

Example: True, False
