2.2 - Variables and Constants


2.2.1 - Know the definition of variables, understand their purpose and when they are used.

Definition: A variable is a named storage location in memory that holds a value which can change during program execution.

Purpose: Store data temporarily, allow programs to process inputs, perform calculations, and store outputs.

When Used: Anytime the program needs to work with values that may change, such as user input, calculations, or counters.

Example: age = 16, age = age + 1, print(age) → 17

2.2.2 - Know the definition of constants, understand their purpose and when they are used.

Definition: A constant is a named value in memory that cannot change during program execution.

Purpose: Improve readability (e.g., PI instead of 3.14159) and prevent accidental changes to important values.

When Used: When a value is fixed and should not be modified.

Example: PI = 3.14159, radius = 5, area = PI * radius**2

2.2.3 - Understand the purpose of data type conversion functions and why they are used. See Appendix 2 for functions.

Purpose: Convert data from one type to another, so it can be used correctly in operations.

Why Used: To avoid type errors (e.g., trying to add a string to an integer) and to match input/output requirements.

Common Functions: int() → convert to integer, float() → convert to decimal, str() → convert to string

Example: user_input = input("Enter your age: "), age = int(user_input), print(age + 1)

2.2.4 - Know a definition of scope and understand the role of scope and when it is used.

Definition: Scope refers to the region of a program where a variable can be accessed.

Role of Scope: Controls visibility and lifetime of variables, preventing unwanted interference between parts of code.

When Used: Anytime variables are declared, they exist within a scope (e.g., inside a function or globally).

2.2.5 - Know and understand how variables are managed by scope: global variables, local variables.

Global Variables

Definition: Declared outside functions and accessible anywhere in the program.

Local Variables

Definition: Declared inside functions and only accessible within that function.

Example:

x = 10  # global

        def my_function():
            y = 5  # local
            print(x + y)

        my_function()   # Works: prints 15
        print(x)        # Works: prints 10
        # print(y) → ERROR (y is local)

2.2.6 - Be able to use scope.

You must be able to understand how to use scope.

2.2.7 - Be able to declare variables and constants using standard data types.

You must be able to declare variables and constants using standard data types (e.g., integer, float, string, boolean).

2.2.8 - Be able to use variables and constants.

You must be able to use variables and constants effectively within a program.

2.2.9 - Be able to use data type conversion functions.

You must be able to use data type conversion functions (e.g., int(), float(), str()).
