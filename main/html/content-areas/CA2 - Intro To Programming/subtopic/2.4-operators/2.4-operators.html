2.4 - Operators


2.4.1 - Understand the purpose of arithmetic operators

Purpose: Perform mathematical calculations on numbers.

Operators:

  • Add: +
  • Subtract: -
  • Multiply: *
  • Divide: /
  • Exponentiation: **
  • Integer Division: //
  • Modulus: %

Example:


a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.3333...
print(a ** b)  # 1000
print(a // b)  # 3
print(a % b)   # 1

2.4.2 - Understand the purpose of relational operators

Purpose: Compare values and return True or False.

Operators:

  • Equivalence: ==
  • Less than: <
  • Greater than: >
  • Not equal: !=
  • Less than or equal: <=
  • Greater than or equal: >=

Example:


x = 7
y = 10

print(x == y)  # False
print(x < y)   # True
print(x != y)  # True

2.4.3 - Understand the purpose of Boolean operators

Purpose: Combine or invert Boolean values (True/False).

Operators:

  • Not: not
  • And: and
  • Or: or

Example:


a = True
b = False

print(not a)      # False
print(a and b)    # False
print(a or b)     # True

2.4.4 - Be able to use operators

Operators can be used to perform calculations, compare values, and control logic in programs.

Example:


x = 5
y = 10

# Arithmetic
sum = x + y  # 15

# Relational
is_equal = x == y  # False

# Boolean
result = (x < y) and (y > 5)  # True

2.4.5 - Be able to interpret code using operators

When reading code, identify the operator, the values it works on, and predict the result.

Example:


x = 4
y = 6
print(x * 2 == y)  # False

2.4.6 - Be able to create code using operators

Practice writing code using arithmetic, relational, and Boolean operators.

Example Challenge:


# Check if a number is divisible by 3 but not by 5
num = 9
if (num % 3 == 0) and (num % 5 != 0):
    print("Yes")
else:
    print("No")

2.4.7 - Be able to debug code using operators

Common errors when using operators:

  • TypeError: Mixing incompatible types. 
    
x = 5
y = "10"
print(x + y)  # ERROR
  • Comparison error: Using = instead of ==. 
    
a = 5
b = 10
print(a = b)  # ERROR
  • Boolean logic mistakes: Forgetting parentheses. 
    
x = 5
y = 10
z = 15
print(x < y and y < z)  # Correct
