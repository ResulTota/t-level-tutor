2.6 - Actions


2.6.1 - Know and understand sequence

Definition: Sequence is the order in which instructions are executed in a program, from top to bottom.

Purpose: Ensure instructions are executed logically and predictably.

When Used: In any program where actions must happen step by step.

Example:


print("Start")
x = 5
y = x + 2
print("Result:", y)
print("End")

2.6.2 - Know and understand selection

Definition: Selection allows a program to choose between different actions based on conditions.

Purpose: Make decisions in programs.

When Used: When a program needs conditional logic, e.g., deciding grades or menu options.

Keywords:

  • if
  • elif (else if)
  • else
  • match/case

Example with if/elif/else:


score = 75

if score >= 90:
    print("Grade A")
elif score >= 75:
    print("Grade B")
else:
    print("Grade C or below")

Example with match/case (Python 3.10+):


day = "Monday"

match day:
    case "Monday":
        print("Start of week")
    case "Friday":
        print("End of week")
    case _:
        print("Midweek")

2.6.3 - Know and understand loops

Definition: Loops repeat a set of instructions multiple times.

Purpose: Avoid repeating code manually and automate tasks.

Types:

  • Count-controlled loops: repeat a fixed number of times
  • Condition-controlled loops: repeat while a condition is true

Example (count-controlled for loop):


for i in range(5):
    print("Iteration", i)

Example (condition-controlled while loop):


x = 0
while x < 5:
    print("x is", x)
    x += 1

2.6.4 - Know and understand iteration

Definition: Iteration is the process of repeating instructions using loops.

Count-controlled iteration:


for i in range(10):
    print(i)

Condition-controlled iteration:


n = 0
while n < 10:
    print(n)
    n += 1

2.6.5 - Know and understand the benefits and drawbacks of loops

Benefits:

  • Reduces code duplication
  • Makes programs easier to maintain
  • Automates repetitive tasks

Drawbacks:

  • Can cause infinite loops if conditions are wrong
  • Can be inefficient if used unnecessarily
