2.8 - Validation


2.8.1 - Definition, Purpose, and Types of Validation

Definition: Validation is the process of ensuring that the data entered into a program or system is correct, reasonable, and acceptable before it is processed.

Purpose: Prevent invalid data, ensure accurate processing, and improve user experience.

Presence Check

Ensures a value is entered and not left blank.


        username = input("Enter username: ")
        if username == "":
            print("Error: Username required")

Length Check

Ensures input is within a specific length range.


        password = input("Enter password: ")
        if len(password) < 8:
            print("Error: Password too short")

Range Check

Ensures numbers are within a valid range.


        age = int(input("Enter your age: "))
        if age < 1 or age > 120:
            print("Error: Invalid age")

Type Check

Ensures the input is of the correct data type.


        try:
            age = int(input("Enter your age: "))
        except ValueError:
            print("Error: Age must be a number")

Format Check

Ensures input follows a specific pattern or format.


        import re
        email = input("Enter your email: ")
        if not re.match(r"[^@]+@[^@]+\.[^@]+", email):
            print("Error: Invalid email format")

Check Digit

Validates numbers using a calculated extra digit, commonly used in barcodes or ISBN numbers.


        # Example of a simple check digit validation for a 4-digit number
        number = "1234"
        check_digit = (int(number[0]) + int(number[1]) + int(number[2])) % 10
        if check_digit != int(number[3]):
            print("Error: Invalid check digit")
