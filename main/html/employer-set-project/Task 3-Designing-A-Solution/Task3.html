ESP - Task 3: Designing A Solution

What Is Task 3?

Task 3 is all about making Flow Charts.
You will be given a brief and a CSV file that your Flow Chart(s) will be based on.

CSV Files

What are CSV files..?
A CSV (Comma Separated Values) file is a plain text file that contains data in a tabular format.
Each line in the file represents a row of data, and each value within the row is separated by a comma.
Within the ESP, the CSV file will contain data relevant to the brief.
Note: You will not receive the same CSV file in Task 4a, however it is highly likely that the CSV file will have similar data.

  • Almost every CSV file you will be given will have a form of Time

  • Several columns with unique headers
  • Each value (outside of Day) is unique

You will use this data to create your Flow Chart(s), as you will need to reference the data in the CSV file to meet the brief requirements.

Flow Charts

Soooo... What are Flow Charts..?
A flow chart is a type of diagram that represents an algorithm.
Flowcharts can:

  • Communicate complex ideas in a simple, graphical way
    (They can act as a bridge between a programmer and a client, to ensure everyone knows what is expected of a program)
  • Show the steps in a process as shapes
  • Help to understand how a process works
  • Identify potential problems or inefficiencies
  • Design processes or improve existing ones

To make your flow chart, you will likely use a software called: draw.io

  • Easy to follow

Marks

Task 3 is worth 17 marks.
The marks are split into 3 parts:

  • Decomposition of the Probelm - 8 marks
  • Application of logical thinking and conventions - 6 marks
  • Communication of the design - 3 marks

Time

You will have 3 Hours to complete this task.

Tips

  • Spend time getting familiar with draw.io before the exam.
  • Make sure to read the task brief carefully and ensure you understand the requirements before beginning anything else.
  • Use the data in the CSV file to help you meet the requirements of the brief.
  • Do not overcomplicate things, keep your flow chart simple and easy to follow.
